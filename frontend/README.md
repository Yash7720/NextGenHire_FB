# 🚀 NextGenHire — Gamified Learning & Skill-Based Hiring Platform

A full-featured React + Vite + Tailwind application.

---

## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev

# 3. Build for production
npm run build
```

---

## Dependencies

| Package | Purpose |
|---|---|
| `react-router-dom` | Client-side routing |
| `recharts` | Charts (bar, radar, pie) |
| `lucide-react` | Icons (optional, installed but not required) |
| `clsx` | Conditional classNames |
| `tailwindcss` | Utility CSS |
| `@vitejs/plugin-react` | Vite React plugin |
